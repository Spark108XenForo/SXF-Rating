# [SXF] Rating
Page with different ratings.

**Realized ratings:**
* Number of resources (Requires XenForo Resource Manager).
* Number of sympathies.
* Number of messages.
* Number of points.

You can also create your own ratings. [/docs/manual/eng_create_rating.md](How to create ratings?)
